Penulisan Parameter Return Keterangan
() => "Hi" ❌ (Tiada) ✅ Auto-return Paling ringkas, return string terus
a => a * 2 ✅ 1 parameter ✅ Auto-return Boleh tanpa () bila 1 parameter
(a, b) => a + b ✅ 2+ parameter ✅ Auto-return Diguna untuk operasi ringkas
a => ({nama: a}) ✅ 1 parameter ✅ Return object Mesti bungkus object dalam ()
() => ({status: "ok"}) ❌ ✅ Return object Tanpa parameter, return object literal
(a, b) => { return a + b; } ✅ ❌ Manual return Block body perlukan return
() => { console.log("test"); } ❌ ❌ Tiada return Untuk side-effect sahaja
(a, b) => { const x = a * b; return x + 1; } ✅ ❌ Manual return Diguna jika logic lebih dari 1 baris
() => 123 ❌ ✅ Return nombor Return number terus
() => [1, 2, 3] ❌ ✅ Return array Return array secara terus
() => null ❌ ✅ Return null Return nilai khas/null
