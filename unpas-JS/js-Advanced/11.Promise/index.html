AJAX ialah singkatan untuk Asynchronous JavaScript and XML.

AJAX ialah teknik dalam JavaScript yang digunakan untuk: Mengambil atau menghantar data ke server tanpa reload halaman (tanpa refresh page).

⚡ Apa Itu Promise? Promise adalah objek khas dalam JavaScript yang mewakili hasil masa depan (akan datang) dari operasi asynchronous. Promise ada 3 keadaan: Status Maksud pending Belum selesai fulfilled Berjaya rejected Gagal (error)

fetch() adalah contoh fungsi yang return Promise then() dan catch() digunakan untuk tunggu hasil async/await digunakan untuk mudahkan tulis kod yang tunggu Promise

resolve() bermaksud janji berjaya — akan dihantar ke .then().

.then() digunakan selepas promise berjaya.

Sekarang, kita tukar cara then() jadi async/await (lebih kemas).

| Baris | Maksud | | ------------------ | ------------------------------- | | `async function` | Buat function async | | `await fetch(...)` | Tunggu sampai fetch siap | | `await res.json()` | Tunggu hasil convert JSON | | `try...catch` | Tangkap error kalau ada masalah |
